# AsyncDisplayKit (Texture) — XCFramework via Swift Package Manager

This repository provides **AsyncDisplayKit (also known as Texture)** packaged as an **XCFramework** with support for Swift Package Manager (SPM).

AsyncDisplayKit / Texture is an iOS framework for building smooth, responsive user interfaces.  
For more details, documentation, and source code, please refer to the official repository:  
👉 [https://github.com/TextureGroup/Texture](https://github.com/TextureGroup/Texture)

---

## 📦 Installation via Swift Package Manager

Add the package to your project in Xcode:

1. Open your project in **Xcode**.
2. Go to **File > Add Packages…**
3. Enter this repository URL: https://github.com/resoul/AsyncDisplayKit.git

---

## 🛠 Usage

In your Swift/Objective-C code:

```swift
import AsyncDisplayKit
```
or in Objective-C:
```swift
@import AsyncDisplayKit;
```
Supported Platforms

iOS (iPhone, iPad)

Notes

This package distributes prebuilt XCFrameworks for iOS and tvOS.
